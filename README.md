# Projet_PPC
## Pour démarrer la simulation:
1. Ouvrir le code main.py et lancer un nouveau terminal.
2. Démarrer la simulation en tapant la commande '''python main.py'''.
3. Des messages d'initialisation s'affichent, veuillez paramètrer la simulation (quantité d'herbe, nb de proies, nb de prédateurs, duration de la secheresse).
4. Après avoir parametré la simulation, une fênetre d'affichage pygame apparaît et vous pourrez visualizer l'environnement avec les paramètres choisis et un tableau de bord qui affiche les quantités de proies et predateurs actives, passives, les naissances et les dèces.

## Pendant la simulation:
Vous pouvez déclencher une sécheresse manuelle en appuyant sur la touche [D] à n'importe quel instant.
Vous pouvez également quitter la simulation à n'importe quel instant en appuyant sur la touche [Q].

## Après la simulation:
Un bilan final est affiché avec la quantité d'herbe, proies et prédateurs restants, accompagné d'un graphique d'évolution temporelle des espèces présentes dans l'environnement et un message qui spécifie le mode d'arrêt de la simulation (automatique ou manuel). Appuyez sur n'importe quelle touche pour quitter.

Bonne simulation!
